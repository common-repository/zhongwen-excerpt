中文摘要插件。

== Description ==

弥补大部分国外主题未考虑的中文字符截断设置。用户可自定义HTML标签以及CSS链接样式，从而最大化个性设置。插件绿色环保，停用后自动初始化并清除相关数据库表单。

由于WP官网部分内容被墙，未翻墙的用户可以直接到插件官网看截图：http://www.bianlei.com/zhongwen-excerpt/

== Installation ==

1. 在WordPress管理后台，点击左侧的“插件”按钮，然后点击“安装插件”，进入在线安装插件页面，在搜索框中输入“Zhongwen Excerpt”，按键盘回车搜素，找到插件并点击“现在安装”。
2. 在WordPress管理后台，点击左侧的“插件”按钮，找到名称为“Zhongwen Excerpt”的插件并点击“启用”。
